Interface d’onboarding client premium inspirée de Typeform et Tally, conçue pour fluidifier les demandes de transport de marchandises pour Ngwéa.

✨ Aperçu

Ce projet est un formulaire multi-étapes ultra-moderne développé en HTML, CSS et JavaScript vanilla.
Il offre une expérience utilisateur immersive avec :

une seule question à la fois,
des transitions fluides,
une navigation clavier intuitive,
un design responsive élégant,
une soumission asynchrone via Formspree.

L’objectif est de simplifier la collecte des besoins logistiques clients tout en offrant une expérience haut de gamme.

🚀 Fonctionnalités
🎯 UX inspirée de Typeform / Tally
📱 Design 100% responsive
⚡ Navigation fluide avec la touche Entrée
📊 Barre de progression dynamique
🎨 UI moderne avec animations
🟢 Affichage conditionnel intelligent du budget
📤 Soumission asynchrone via Fetch API + Formspree
✅ Écran de confirmation personnalisé
🖼️ Intégration du branding Ngwéa
🛠️ Stack Technique
HTML5
CSS3
JavaScript Vanilla
Formspree API
